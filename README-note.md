
# README-NOTE: Landing Page сервисного центра «Дройд»

### Описание проекта
Одностраничный сайт (Landing Page) для сервисного центра по ремонту мобильных устройств «Дройд». Проект создан с использованием чистого HTML5 и CSS3 без применения JavaScript. Дизайн выполнен в сдержанном, технологичном стиле с тёмной цветовой схемой и синими акцентными элементами, ориентирован на мужскую аудиторию 30-40 лет. Основной упор сделан на функциональность, доверие и удобство пользователя.

---

### Итерации и ключевые изменения

На протяжении разработки было реализовано несколько ключевых улучшений и добавлений:

#### 1. Начальная версия
- Создана полная HTML-структура сайта с семантической разметкой.
- Реализованы все основные блоки: шапка, первый экран с формой, преимущества, виды неисправностей, прайс-лист, отзывы, примеры работ, контакты и футер.
- Добавлена плавающая кнопка «Заказать звонок» для быстрой связи.
- Реализованы базовые слайдеры для прайс-листа, отзывов и примеров работ на чистом CSS с использованием `input[type=radio]`.
- Настроена адаптивность для десктопов и планшетов (включая разрешение 1024x1366).

#### 2. Улучшение слайдера «Прайс-лист»
- Переработана анимация смены слайдов: вместо простого появления реализован плавный горизонтальный сдвиг (`translateX`).
- Зафиксирована высота блока для предотвращения "прыжков" контента при смене слайдов.
- Улучшена стилизация навигационных точек.

#### 3. Добавление блока «Контакты» с картой
- Блок разделен на две колонки: слева — контактная информация, справа — интерактивная карта Google.
- Для соответствия общему дизайну фон блока изменён на тёмный (`#16213e`).
- Реализована адаптивность: на мобильных устройствах карта располагается под контактной информацией.
- Предоставлена инструкция по встраиванию актуального iframe-кода Google Maps.

#### 4. Переработка блока «Отзывы»
- Горизонтальный слайдер заменён на эффектный **вертикальный слайдер**.
- Реализована анимация "вращения" отзывов сверху вниз при переключении.
- Активный отзыв подсвечивается акцентной рамкой и тенью, становясь визуально ближе к пользователю.
- Неактивные отзывы стилизованы (уменьшены, без рамки и тени), чтобы не отвлекать внимание.
- Решена проблема обрезки увеличенного активного слайда с помощью правильной настройки `overflow`.

#### 5. Унификация слайдера «Примеры работ»
- Слайдер переделан по аналогии с улучшенным «Прайс-листом».
- Реализован плавный горизонтальный сдвиг слайдов с изображениями «Было/Стало».
- Добавлена навигация точками снизу для интуитивного управления.
- Высота блока зафиксирована, стили изображений настроены для сохранения пропорций.

#### 6. Редизайн футера (подвала сайта)
- Структура футера изменена на двухколоночную: слева — основная навигация по сайту, справа — краткое описание компании и юридическая информация.
- Дизайн приведен в соответствие с блоком контактов (тёмный фон, белые и синие акцентные цвета).
- Добавлен эффект при наведении на пункты навигации.
- Реализована адаптивность с вертикальным расположением колонок на малых экранах.

#### 7. Добавление блока «Вакансии»
- Создан новый функциональный блок, расположенный перед футером.
- Структура из двух колонок: слева — список вакансий в виде **CSS-аккордеона**, справа — блок с инструкцией по отклику.
- Аккордеон позволяет скрывать и раскрывать описание вакансий без использования JS, экономя место на странице.
- Правая колонка сделана «липкой» (`position: sticky`) для удобства пользователя на десктопах.

### Технические детали

- **Стек:** HTML5, CSS3.
- **JavaScript:** Не используется. Вся интерактивность (слайдеры, аккордеон) реализована на чистом CSS.
- **Архитектура CSS:** Используются CSS-переменные для легкой смены цветовой палитры и шрифтов. Стили структурированы по блокам.
- **Адаптивность:** Flexbox и Media Queries используются для корректного отображения на десктопах, планшетах и мобильных устройствах.
- **Интерактивные элементы:** Слайдеры и аккордеон построены на связке `input[type="radio/checkbox"]`, `label` и селектора `~`.
- **Шрифты:** Подключен Google Fonts (Roboto).
- **SEO и доступность:** Использована семантическая разметка (`<header>`, `<section>`, `<footer>` и т.д.), прописаны `alt` для изображений (плейсхолдеры).

### Структура файлов

- `index.html` — основной файл разметки, содержащий всю структуру страницы с комментариями к каждому блоку.
- `style.css` — файл стилей, также структурированный и комментированный для удобства навигации.

### Рекомендации по дальнейшей работе

- **Формы:** Для полноценной работы форм обратной связи требуется подключение серверной части (PHP, Node.js, etc.) для обработки и отправки данных.
- **Карта:** Заменить плейсхолдер iframe в блоке контактов на актуальный код встраивания с вашего аккаунта Google Maps.
- **Контент:** Заполнить сайт реальными текстами, фотографиями, отзывами и ценами.
